## 최대공략수 구하기
1. 큰수 a 작은수 b를 먼저 설정
2. a에서 b를 나눈 나머지가 작은수가 된다.
3. 기존의 작은 수는 큰수가 된다.
 - a,b -> old a는 old b로변경 -> new b는 old a % old b 로 변경
4. 작은 수가 0이 될때 까지 반복하며 a가 최대공략수가 된다
```python
    while b!=0:
        temp = a % b
        a = temp
        b,a = a,b
```
